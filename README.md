# Sprint_4

### Покрытие тестами класса BooksCollector

1. **add_new_book**
   - добавление книги с валидным названием
   - проверка добавления книги с невалидным названием
   - проверка добавления нескольких книг

2. **set_book_genre**
   - установка валидного жанра существующей книге
   - проверка установки несуществующего жанра
   - проверка установки жанра несуществующей книге

3. **get_book_genre**
   - получения жанра существующей книги
   - проверка получения жанра книги без жанра
   - проверка получения жанра несуществующей книги

4. **get_books_with_specific_genre**
   - получение списка книг по жанру
   - проверка получения книг по несуществующему жанру
   - проверка получения книг по жанру, в котором нет книг

5. **get_books_genre**
   - проверка вывода + фоном проверяется в других тестах

6. **get_books_for_children**
   - проверка, что возвращаются только книги с подходящими детям жанрами

7. **add_book_in_favorites**
    - добавление книги в избранное
    - проверка добавления несуществующей книги в избранное
    - проверка добавления книги в избранное дважды

8. **delete_book_from_favorites**
    - удаление книги из избранного
    - проверка удаления из избранного несуществующей книги

9. **get_list_of_favorites_books**
    - проверка вывода + фоном проверяется в других тестах

### Запуск тестов
```bash
pytest -v tests.py
```
